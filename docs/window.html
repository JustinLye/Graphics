window.lib

The window library consist of jlg_window.h.

jlg_window.h

The jlg_window header file declares clear_color struct and Window class. The Window class is defined in jlg_window.cpp.

clear_color struct

The primary function of the clear_color struct is to encapsulate RGBA values.

The clear_color struct contains four GLfloat members represent Red, Green, Blue, and Alpha values.

Window class

The primary function of the Window class is to serve as an interface to an OpenGL window.

When Window objects are constructed an internal method _create_window() is invoked. This method does several things. First, a static boolean member property of the Window class called _is_initialized is evaluated. If _is_initialized is false, then the method _init() is invoked. This method will initialize the GLFW system. Second, the internal member _window is set to a newly created GLFWwindow* via the glfwCreateWindow() method. Third, a static boolean member propery of the Window class called _is_first_window is evaluated. If _is_first_window is true, then the GLEW system is initialized. Finally, an internal member of the Window class called _is_created is set to the value of the expression _window != nullptr.

It is important to mention the viewport must be manually set with a call to the method SetViewport(). The background color of the window can be changed via the SetColor() method. The method Rendor() will invoke two OpenGL methods glClearColor() and glClear().
